# Sample Website

This folder contains a minimal static website scaffold and a tiny Node static server for local testing.

Run locally:

```powershell
node serve.js
# then open http://localhost:3000 in your browser
```

Files:

- `index.html` — main page
- `styles.css` — simple styles
- `app.js` — small client-side interactions
- `serve.js` — tiny Node static server (for local testing)
